# DevOps Portfolio Website

A modern, responsive personal portfolio website built with **React**, **Tailwind CSS**, and **Vite**, designed to showcase DevOps skills, projects, and automation mindset.  

This project is also used as a **hands-on DevOps learning platform** to demonstrate CI/CD, automated deployments, and cloud hosting.

---

## Live Demo

### Below is the link of my website

Website URL : [Visit my DevOps Portfolio](https://devops-portfolio-saqlain-yaqoob.vercel.app/)

---

## Tech Stack

- **Frontend:** React.js
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Font Awesome / React Icons
- **Version Control:** Git & GitHub
- **Hosting & CI/CD:** Vercel

---

## Features

- Responsive design (mobile, tablet, desktop)
- Clean, minimal, professional layout
- Production-ready build setup
- Resume download functionality

## Local Development

### Install dependencies

- npm install

- Start development server

- npm run dev

- Build for production

- npm run build

- Deployment (Vercel)

## This project is deployed using Vercel, enabling

- Automatic builds on Git push
- Continuous Deployment (CI/CD)
- Global CDN
- HTTPS by default

## DevOps Learning Objectives

### This portfolio is not just a website, it is a DevOps practice project demonstrating

- CI/CD pipelines
- Automated builds and deployments
- Cloud hosting workflows
- Git-based version control
- Production readiness
